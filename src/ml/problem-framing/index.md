# Problem Framing

해당 섹션의 목표는 다음과 같습니다.

- 일반적인 ML 용어의 정의
- ML을 사용하는 상품과 각각에서 쓰이는 ML 문제 해결의 일반적인 방식
- ML로 특정 문제를 해결할 수 있는지를 구분
- 다른 프로그래밍 방식과 ML의 비교 및 대조
- ML 문제들에 대한 가설 검증과 과학적 방법
- ML 문제 해결에 대한 이야기
